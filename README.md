# ACP-to-Letta Bridge

Connect Zed Editor to Letta Agents for stateful AI coding assistance with persistent memory.

## Features

- ✅ Full ACP (Agent Client Protocol) compatibility
- ✅ Persistent memory across sessions via Letta
- ✅ Code completion and editing
- ✅ Tool execution (web search, code execution)
- ✅ Multi-agent support (Bizdev, specialists)
